About This Plugin

The Snowflix plugin provides accessibility features with visual effect filters for disabled users. It offers 6 different viewing modes:

  • Billboard: 3D billboard effect with day/night modes and rotation controls
  • TV: 3D television viewing angle with day/night toggle
  • Flashlight: Spotlight effect with adjustable position and size
  • Desaturation: Adjustable color saturation from B&W to full color
  • Toon: Cartoon/comic book effect with bright lines, dark lines, or color modes
  • RGB: Color filter options (red, green, blue, and combinations)

How to use: Click the power button to activate the plugin, then select a viewing mode from the right panel. Each mode has customizable options.
