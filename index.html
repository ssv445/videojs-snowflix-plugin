Video.js Snowflix Accessibility Plugin

Available Filters

  • Billboard: 3D billboard effect with rotating prisms
  • TV: TV viewing mode with adjustable camera position
  • Flashlight: Spotlight effect following cursor
  • Desaturation: Grayscale filter with adjustable intensity
  • Toon: Cartoon/cel-shading effect
  • RGB: Color channel filters (Red, Green, Blue)

Development Notes

This is the Vite development server. The plugin is now built with:

  • Pure ESM format
  • External assets (not base64 inlined)
  • Organized asset structure in dist/assets/
  • Modern Vite build system following Video.js standards
